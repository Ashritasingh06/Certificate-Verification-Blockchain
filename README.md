# Certificate Verification using Blockchain

This project demonstrates how blockchain can be used to issue and verify certificates securely using Ethereum smart contracts.

## 🔧 Technologies Used
- Solidity
- Remix IDE
- Ethereum (Remix VM – London)
- Blockchain Smart Contracts

## 📄 Smart Contract
CertificateVerification.sol

### Features:
- Issue certificate using certificate ID and hash
- Verify certificate authenticity
- Immutable blockchain storage

## 🧪 How to Run (Demo)
1. Open https://remix.ethereum.org
2. Upload CertificateVerification.sol
3. Compile using Solidity 0.8.x
4. Deploy using Remix VM (London)
5. Call `issueCertificate(certId, certHash)`
6. Call `verifyCertificate(certId, certHash)`

## 📝 Note
This project is for academic and demonstration purposes.  
No real certificates are stored on blockchain.
